# Naranja Feliz — Руководство Пользователя

## Для учеников

### Регистрация и вход
1. Открой сайт → нажми «Зарегистрироваться»
2. Укажи имя, email, пароль → «Создать аккаунт»
3. Подтверди email (письмо придёт на почту)
4. Войди с email и паролем

### Главная страница
После входа ты попадаешь на список доступных курсов. Здесь отображаются:
- **Курсы, на которые ты записан** — с прогрессом (🍊 дольки апельсина)
- **Доступные курсы** — можно записаться (кнопка «Записаться»)

### Прохождение урока
1. Нажми на курс → выбери урок из списка
2. Проходи блоки по порядку:
   - **Текст** — просто читаешь
   - **Изображение** — смотришь картинку
   - **Видео** — смотришь видео (YouTube, Rutube, VK)
   - **Вставка слова** — заполни пропуски, нажми «Проверить»
   - **Выбор ответа** — выбери правильный вариант(ы), нажми «Ответить»
   - **Открытый вопрос** — напиши ответ, нажми «Отправить на проверку» (автосохранение при уходе с поля)
   - **Аудио-ответ** — нажми «🎙 Записать», говори, нажми «⏹ Остановить», прослушай, нажми «Отправить»
   - **Составь предложение** — кликай по словам, чтобы расставить их в правильном порядке, нажми «Проверить»
   - **Выбери изображение** — кликни на нужную картинку(и), нажми «Ответить»
3. Когда выполнишь все интерактивные блоки — урок автоматически отметится как пройденный (🍊 Долька собрана!)

### Чат с учителем
Под каждым отправленным ответом (открытый вопрос, аудио-ответ) есть поле для сообщения. Ты и учитель можете переписываться в этом чате.

### Проверка учителем
Когда учитель проверит твой ответ:
- Ответы отметятся зелёным «✓ Проверено»
- Под ответом появится комментарий учителя

### Уведомления
🔔 Колокольчик в шапке показывает непрочитанные уведомления:
- Учитель проверил ответ
- Учитель оставил комментарий

### Настройки профиля
⚙️ Настройки (в меню аватара):
- Сменить аватар (загрузить фото)
- Сменить имя
- Сменить email
- Сменить пароль
- Посмотреть свою роль и уровень (🍊 Новичок → Профессор)

## Для учителей

### Режим учителя
В шапке есть переключатель «Учитель» / «Ученик». В режиме учителя становятся доступны:
- Пункты меню: «Учителя», «Проверка», «Статистика»
- Твои курсы на главной странице (с кнопкой «+ Создать курс»)

### Создание курса
1. Нажми «+ Создать курс» на главной или в админке
2. Укажи название, описание, уровень (A1–C2)
3. Нажми «Создать курс»

### Управление уроками
1. Зайди в курс → «+ Новый урок»
2. Укажи название и порядковый номер
3. Нажми «Создать» → откроется редактор урока

### Редактор блоков
Добавляй блоки к уроку:
- **Текст** — обычный текст (отображается как есть)
- **Изображение** — вставь ссылку на картинку или нажми «Загрузить»
- **Видео** — вставь ссылку на YouTube, Rutube или VK Video
- **Вставка слова (FillBlank)** — шаблон с пропусками в квадратных скобках, например: «Я [люблю] учить [испанский]»
- **Выбор ответа (Choice)** — варианты через запятую, правильные номера (1, 2, 3)
- **Открытый вопрос** — просто напиши вопрос
- **Аудио-ответ** — задание записать аудио
- **Составь предложение (DragOrder)** — шаблон как во FillBlank: слова в скобках перемешиваются
- **Выбери изображение (ImagePick)** — добавь картинки (URL + подпись), укажи правильные, выбери режим (один/множественный)
- Для изображений в ImagePick и Image: нажми «Загрузить» → выбери файл → URL появится автоматически

Кнопка «Опубликовать» делает урок видимым для учеников.

### Проверка заданий
1. Меню → «Проверка»
2. Отфильтруй по курсу, уроку, студенту (поле поиска)
3. Нажми на ответ → откроется панель с:
   - Текстом ответа
   - Чатом (можно обсудить с учеником)
   - Кнопкой «✓ Принять» (подтвердить проверку)

### Назначение учителей
1. Меню → «Учителя»
2. Введи email пользователя → «Сделать учителем»
3. Пользователь получит права учителя сразу (без перелогина)

### Статистика
Меню → «Статистика» показывает:
- Количество пользователей, курсов, уроков, блоков
- Заполненность хранилища (Supabase Storage, 1 ГБ)
- Заполненность базы данных (500 МБ)

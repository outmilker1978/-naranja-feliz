## 7. Марафоны

**Идея:** Новый тип учебного материала на уровне курсов — «Марафон».

- По структуре как курс: внутри уроки

## 8. Упрощение контента

Категории (`category`) у типов `news`, `article`, `ad` не влияют на отображение:
- **news**: announcement / event / achievement — сырой текст на странице, бесполезно
- **article**: grammar / culture / tips — то же самое
- **ad**: banner / promo — не используется вообще

**Надо:** убрать поле `category` из редактора для этих типов (оставить только для `page_section`), очистить старые записи.

### page_section: дубль «Заголовок» и «Заголовок секции»
Общее поле `title` (столбец) и поле `content.title` (в JSON) дублируются. При рендере на главной используется только `content.title`.

**Надо:** убрать общий `title` из формы для `page_section`, оставить только поля внутри секции.

### Реклама: дубль полей картинки
У `ad` два поля для одного изображения: «Обложка» (общая) и «Изображение (URL)» (в блоке баннера). При рендере `cover_image || content.image`. Одно лишнее.

**Надо:** оставить одно поле — загрузка файла (с ПК) + URL. Убрать дубль.

### Во все поля с URL картинки — добавить загрузку с ПК
Сейчас только «Обложка» умеет загружать файл с компьютера. Остальные поля (реклама → изображение, о школе → фото, отзывы → аватар) — только URL. Не у всех есть ссылка на картинку.

**Надо:** вынести загрузку файла в переиспользуемый компонент и воткнуть во все поля изображений.
